# GitHub Source Link Generation

> **Skill**: Load `skills/github-source-link-generator/SKILL.md` when the user wants a clickable URL to code in a repository.

Use this for:

- linking a code snippet to GitHub or GHE
- generating `blob/<ref>/<path>#Lx-Ly` URLs
- adding clickable source links into diagrams, docs, or review comments
- converting a tracked local file and line range into a remote URL

Prefer commit permalinks when the commit is known to exist on the remote.

This file documents custom commands defined in this config.

==============================================================================
                                                       *:BlobDiff* *BlobDiff*
:BlobDiff {rev} {oldpath} {newpath}

Open a vertical diff between a Git blob and a file on disk.

Arguments: >
  {rev}      Git revision, for example HEAD or HEAD~1
  {oldpath}  Path inside the Git revision
  {newpath}  Path to open from the working tree
<
Behavior:
  - Opens {newpath} in the current window
  - Opens a scratch buffer in a vertical split
  - Loads {rev}:{oldpath} into the scratch buffer
  - Starts diff mode in both windows

Example: >
  :BlobDiff HEAD~1 .config/nvim/init.lua .config/nvim/init.lua
<
Tips:
  - Use |:vert diffsplit| style navigation commands after opening the diff
  - Use your Telescope help picker to find this entry via |BlobDiff|
  - Use your Telescope commands picker to discover the command itself
